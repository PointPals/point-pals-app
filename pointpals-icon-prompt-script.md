# PointPals — Google Imagen Prompt Script

> Paste the **General Style Prompt** once at the top of your generation session, then send each icon line one at a time, attaching the companion reference image where noted.

---

## General Style Prompt (paste before EACH icon)

```
A single icon illustration only — no background tile, no coloured card, no sticker border. Output on a fully transparent background (PNG with alpha channel). Flat-but-dimensional custom icon style — friendly, slightly rounded, chunky illustration with soft gradients and gentle highlights, a soft warm-charcoal outline (not pure black). Soft pastel colour palette — dusty blue, buttercream yellow, sage green, blush pink, lilac, warm sand, seafoam. No text, no letters, no numbers, no watermark, no photorealism. Square canvas, generous padding, centred.
```

## Batch Workflow (Google Imagen / Gemini)

1. **Session setup:** Open Imagen/Gemini and paste the **General Style Prompt** as your first message
2. **Generate references first:** Run each of the 8 companion character reference prompts above. Save all 8 outputs as your companion reference images for step 4.
3. **Bulk chores:** Run the 44 chores + 12 adult chores — each is a self-contained prompt. These batch fastest because no image attachment is needed.
4. **Companion icons:** For the 12 positive behaviours (B1–B12), **paste the General Style Prompt + icon prompt + attach the companion reference image** for each one.
5. **Needs-work + utility:** Run all 10 needs-work + 2 utility icons at the end.

**Tip:** If your generation tool lets you paste a style seed, use it to keep the visual consistent across all 80 icons before doing companion-specific generations.
